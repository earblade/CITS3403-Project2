Why should I learn React?

React is a powerful tool used to seamlessly and efficiently create code that looks and feels great for the end-user

Ok, but how efficient is it?

React utilises component-based coding, which means that you can you can create complex UIs by composing various components that manage their own states

React also allows you to easily create simple views for each state in your application, updating whenever a change is detected and only changing what it needs to change, preventing long loading times and making the end-user wait

This sort of code makes it easy for debugging as well, since everything is declarable and the problem becomes predictable

It sounds good... But what then what is this site for?

Well, Learn React. is a simple site that will allow you to learn a few important concepts of React that will kickstart your journey in becoming comfortable in coding React

As well as teaching important concepts, we also run you through a multiple choice at the end to test your knowledge, the result of which will be recorded and can be looked over at any time
